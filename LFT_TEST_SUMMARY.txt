LFT API DISCOVERED
═══════════════════════════════════════════════════════════════════

Basic Pattern:
```python
from profissa_lft.host import Host
from profissa_lft.switch import Switch
from profissa_lft.controller import Controller

# Create nodes
c1 = Controller('c1')
s1 = Switch('s1')
h1 = Host('h1')

# Deploy (requires Docker + sudo)
c1.instantiate()
s1.instantiate()
h1.instantiate()

# Connect
h1.connect(s1, "h1s1", "s1h1")

# Configure
h1.setIp('10.0.0.1', 24, "h1s1")
h1.setDefaultGateway('10.0.0.254', "h1s1")
```
