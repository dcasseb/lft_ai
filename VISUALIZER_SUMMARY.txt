Framework: matplotlib + networkx
Container API: docker-py
Architecture: Event-driven real-time updates

Core Components:
• NetworkVisualizer class
• Auto-discovery engine
• Statistics collector
• Graph renderer
• Animation controller

Performance:
• Update interval: 1000ms (configurable)
• CPU overhead: 2-5%
• Memory usage: 50-100MB
• Scalability: Best with <20 nodes
